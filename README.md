## bnbTrack

Income balance tracking tool with Meteor + MongoDB

### System Requires
* Meteor 1.2.1
* Mongo DB 3.0.7

### Setup
* Installing Meteor

```
   curl https://install.meteor.com/ | sh
   sudo -H npm install -g meteorite
```

* Router setting

```
    mrt add router
```

* Adding fullcalendar package

```
    mrt add fullcalendar
```

* Start meteor server

```
    meteor
```
